Job Search Toolkit

Tools

Live jobs

Pulls live Irish roles from Reed.co.uk. New roles (ones you haven't seen before) are highlighted with a green border. Hit "+ Track" on anything worth following up. Salaries are shown in GBP (£) — Reed is a UK job board.

Choose a search mode and hit Search to pull live jobs.

My tracker

Roles you've saved from the live search. Update status as you progress and add notes. Saved in your browser — persists between sessions.

Company tracker

Track target companies, save application status, and keep useful notes for each employer.

Import format

CSV — first row must be a header. Required columns: name, url. Optional: location, tags, status, usefulInfo, lastClicked, lastUpdated. Tags are pipe-separated: EM|DevOps.

JSON — a top-level array of objects with the same field names. Required: name and url. All other fields are optional.

Rows missing name or url are skipped. Companies with a name that already exists are not overwritten. Use Export CSV or Export JSON to see the exact format.

Google alert strings

Set these up at google.com/alerts. Set frequency to "as it happens". Copy each string exactly as shown.

Tip: Google Alerts catches roles posted on company blogs, press releases, and smaller boards that don't always surface on LinkedIn or Indeed.

Job fit scorer

Paste a job description and get a fit analysis against your profile. Customize your profile summary in config.js.

Requires internet connection · Uses Anthropic API · Covered by your Claude subscription


Activity summary

Generate a copyable log of all your update cards within a date range, grouped by company. Handy for INTREO meetings, weekly reviews, or personal records.
